# 타이타닉 생존자 예측 추천 알고리즘 Top 5

캐글(Kaggle) 타이타닉 데이터셋은 크기가 비교적 작고(약 891행) 정형(Tabular) 데이터 형태를 띠고 있습니다. 따라서 복잡한 딥러닝보다는 정형 데이터 분류에서 압도적인 성능을 내는 **앙상블(Ensemble) 및 부스팅 계열 알고리즘**이 가장 높은 정확도를 보여줍니다. 

수많은 캐글러들의 검증 스코어를 바탕으로 정확도가 가장 높게 나타나는 알고리즘 5가지를 정리했습니다.

---

## 1. XGBoost (Extreme Gradient Boosting)
* **특징:** 캐글의 정형 데이터 시합을 지배해 온 대표적인 그라디언트 부스팅 알고리즘입니다. 성능이 매우 강력하며 과적합을 방지하기 위한 규제(Regularization) 매개변수가 잘 갖춰져 있습니다.
* **타이타닉에서의 활용:** 피처 엔지니어링이 정교하게 결합되었을 때 상위권 스코어를 내기 위해 가장 먼저 선택되는 모델 중 하나입니다.

## 2. LightGBM (Light Gradient Boosting Machine)
* **특징:** 마이크로소프트가 개발한 고성능 부스팅 알고리즘입니다. 트리가 깊어지는 형태가 아니라 리프 노드 중심으로 확장하기 때문에, 복잡한 비선형 관계를 매우 빠르고 예리하게 잡아냅니다.
* **타이타닉에서의 활용:** 데이터셋 크기가 작아 자칫 과적합(Overfitting)이 나기 쉽지만, 하이퍼파라미터(`max_depth`, `learning_rate` 등)를 조금만 튜닝해 주면 최고 수준의 정확도를 뿜어냅니다.

## 3. CatBoost (Categorical Boosting)
* **특징:** 타이타닉 데이터셋에는 성별(`Sex`), 탑승 항구(`Embarked`), 호칭(`Title`) 등 **범주형(Categorical) 변수**가 핵심 피처로 작용합니다. CatBoost는 이러한 범주형 변수를 원-핫 인코딩 등 별도의 처리 없이 내부적으로 완벽하게 처리해 주는 특화형 알고리즘입니다.
* **타이타닉에서의 활용:** 기본 설정값(Default)만으로도 전처리 실수 없이 아주 안정적이고 뛰어난 초기 정확도를 보장합니다.

## 4. 랜덤 포레스트 (Random Forest)
* **특징:** 여러 개의 결정 트리(Decision Tree)를 만들어서 투표(다수결)를 통해 최종 생존 여부를 결정하는 배깅(Bagging) 방식의 대표적인 앙상블 모델입니다.
* **타이타닉에서의 활용:** 입문자부터 전문가까지 타이타닉 예제에서 **'가장 가성비 좋고 든든한 모델'**로 꼽힙니다. 데이터에 노이즈가 있거나 결측치가 완전히 정돈되지 않은 상태에서도 과적합이 잘 일어나지 않고 안정적인 성능을 유지합니다.

## 5. 보팅 분류기 (Voting Classifier / Ensemble)
* **특징:** 단일 알고리즘이 아니라, 위에 언급된 우수한 모델들(예: 랜덤 포레스트 + XGBoost + LightGBM 등)을 한 팀으로 묶어 **"집단지성"**으로 예측하는 방식입니다.
* **타이타닉에서의 활용:** 캐글 타이타닉 리더보드에서 상위 1% 이내의 초고득점을 기록한 노트북들은 대부분 단일 모델 하나만 쓰지 않고, 여러 우수한 모델들의 예측치를 소프트 보팅(Soft Voting, 확률 평균)으로 묶어서 최종 제출 파일을 생성합니다.

## 💡 고득점을 위한 핵심 팁
타이타닉 예제에서는 어떤 알고리즘을 쓰느냐보다, **"이름(Name)에서 호칭을 추출하고, 가족 수(`FamilySize`)를 조합하는 등 피처 엔지니어링을 얼마나 정교하게 했는가"**가 정확도를 가르는 80% 이상의 요인으로 작용합니다. 잘 정제된 피처 위에 **XGBoost나 랜덤 포레스트**를 얹고 하이퍼파라미터 튜닝(`GridSearchCV`)을 진행하는 방향을 추천합니다.
